This project is just a template to learn a little bit more about Bulma components. I was bored one night and wanted to try something different than Bootstrap. Therefore you will see huge amounts of placeholder text below.

As you can simply guess, nothing in this page really works, there is no real design or anything. But that was never the intend anyway. There are just some random components styled minimally to check them out. Wanted to see if Bulma components looked good and if the integration with Angular was fine. This entire project has its own repository just because I wanted this somewhere I can access when, or if, I ever want to use this CSS Framework.
